# الفصل 3: الإنتاج، التكاليف، والربح

## 1) أساسيات
- **الإيراد الكلي (TR):** \(TR = P 	imes Q\)
- **التكلفة الكلية (TC):** \(TC = FC + VC\)
- **الربح (\(\pi\)):** \(\pi = TR - TC\)

## 2) متوسطات وتكلفة حدّية
- \(AFC = FC/Q\)
- \(AVC = VC/Q\)
- \(ATC = TC/Q\)
- \(MC = \Delta TC/\Delta Q\)

## 3) تعظيم الربح (منافسة كاملة)
القاعدة: \(MR = MC\). وفي المنافسة الكاملة \(MR = P\).

## 4) شرط الإغلاق (مدى قصير)
إذا كان \(P < AVC\) قد يكون الإغلاق أفضل.
